🛠️ Automasi Metrik Web & Bot CTR

Infrastruktur ini dirancang untuk mensimulasikan trafik secara organik dari Search Engine (Google) menuju target URL dengan menggunakan kontrol parameter secara penuh.

Catatan Penggunaan: Gunakan proxy residensial premium. Menggunakan proxy gratis/publik akan sering menyebabkan kegagalan koneksi atau deteksi anti-bot dari Google.

Fitur Utama

  • Stealth Mode: Memanipulasi sidik jari browser menggunakan playwright-stealth.
  • Anti-Bounce Rate: Skrip akan menelusuri halaman utama dan otomatis mencari link internal untuk diklik menuju sub-page.
  • Rotasi Perangkat: Automasi User Agent (Mobile/Desktop) & Viewport.
  • Auto Proxy Checker: Menyaring dead proxy sebelum bot memakan memori sistem.

Cara Menjalankan

1. Instal semua library yang dibutuhkan:

pip install -r requirements.txt
playwright install chromium

2. Atur parameter di config.json dan masukkan IP di proxies.txt.

3. Eksekusi manajer sistem:

python runner.py
